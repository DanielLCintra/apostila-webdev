Nesta aula daremos continuidade ao estudo de React iniciado anteriormente. Vamos explorar com mais profundidade a sintaxe do JSX, criar componentes funcionais reutilizáveis e aprender como transmitir dados entre eles por meio das props.

1. Sintaxe Avançada do JSX

O JSX permite inserir expressões JavaScript dentro de chaves {} e agrupar elementos com <React.Fragment> ou <>. 

const nome = 'Ana';
function Hello() {
  return (
    <>
      <h1>Olá, {nome}!</h1>
      {nome === 'Ana' && <p>Que bom ver você aqui!</p>}
    </>
  );
}

Comentários em JSX ficam entre {/* */} e é comum usar o operador && ou o ternário para renderização condicional.

2. Componentes Funcionais com Arrow Functions

Podemos declarar componentes como funções tradicionais ou como arrow functions. Essa sintaxe costuma ser mais concisa e facilita a desestruturação de props. 

const Botao = ({ texto }) => <button>{texto}</button>;

export default function App() {
  return (
    <div>
      <Botao texto="Salvar" />
      <Botao texto="Cancelar" />
    </div>
  );
}

Reutilizando o componente Botao com textos diferentes, mantemos o código organizado e fácil de manter.

3. Passando Dados com Props

As props são parâmetros recebidos pelos componentes. Podemos desestruturá-las na assinatura e definir valores padrão. 

function Saudacao({ nome = 'Visitante' }) {
  return <h2>Olá, {nome}!</h2>;
}

export default function App() {
  return (
    <>
      <Saudacao nome="Maria" />
      <Saudacao />
    </>
  );
}

Quando nenhuma prop é enviada, é utilizado o valor padrão Visitante.

4. Composição com children

Componentes podem envolver outros elementos usando a prop especial children. Dessa forma criamos blocos reutilizáveis de layout. 

function Card({ children }) {
  return <div className="card">{children}</div>;
}

export default function App() {
  return (
    <Card>
      <h2>Título</h2>
      <p>Conteúdo enviado como children.</p>
    </Card>
  );
}

Utilizando children é possível compor telas complexas de forma simples e modular.

5. Renderizando Listas

Para criar vários elementos a partir de um array, utilize o método map(). 

const itens = ['React', 'Vue', 'Angular'];
export default function App() {
  return (
    <ul>
      {itens.map(framework => (
        <li key={framework}>{framework}</li>
      ))}
    </ul>
  );
}

6. Hierarquia de Componentes

Em React, os componentes se organizam em uma árvore. Componentes "pais" podem renderizar componentes "filhos" e transmitir dados através das props. 

function Header() {
  return <h1>Meu App</h1>;
}

function Layout({ children }) {
  return <div className="layout">{children}</div>;
}

export default function App() {
  return (
    <Layout>
      <Header />
      <p>Conteúdo principal</p>
    </Layout>
  );
}

Manter uma hierarquia clara facilita a manutenção e ajuda a evitar o props drilling, quando muitas props precisam atravessar vários níveis da árvore.
