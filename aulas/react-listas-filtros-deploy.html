2. Listas Dinâmicas com map()

2.1 Conceito do map()

O método map() é fundamental para renderizar listas em React. Ele transforma cada elemento de um array em um elemento JSX.

2.2 Sintaxe Básica

// Sintaxe básica do map()
const lista = [1, 2, 3, 4, 5];

const elementosJSX = lista.map((item, index) => {
  return <li key={index}>{item}</li>;
});

// Renderização
return <ul>{elementosJSX}</ul>;

2.3 Exemplo Prático - Lista de Contatos

// Componente ContactList
export default function ContactList({ contacts, onRemove }) {
  return (
    <div className="mt-6">
      <h3 className="text-lg font-semibold text-gray-900 mb-4">Lista de Contatos</h3>
      <ul className="space-y-2">
        {contacts.map((contact) => (
          <ContactItem 
            key={contact.id} 
            contact={contact} 
            onRemove={onRemove} 
          />
        ))}
      </ul>
    </div>
  );
}
Importante: Sempre use uma key única para cada elemento da lista. Isso ajuda o React a identificar e atualizar elementos eficientemente.

3. Persistência com localStorage

3.1 Conceito do localStorage

O localStorage é uma API do navegador que permite armazenar dados no navegador do usuário. Os dados persistem mesmo após fechar e reabrir o navegador.

3.2 Métodos Principais

// Salvar dados
localStorage.setItem('chave', 'valor');

// Recuperar dados
const valor = localStorage.getItem('chave');

// Remover dados
localStorage.removeItem('chave');

// Limpar tudo
localStorage.clear();

// Verificar se existe
if (localStorage.getItem('chave')) {
  // dados existem
}

3.3 Trabalhando com Objetos

Como o localStorage só armazena strings, precisamos converter objetos para JSON: 

// Salvar array de objetos
const contatos = [
  { id: 1, nome: 'João', email: 'joao@email.com' },
  { id: 2, nome: 'Maria', email: 'maria@email.com' }
];

localStorage.setItem('contatos', JSON.stringify(contatos));

// Recuperar array de objetos
const contatosSalvos = JSON.parse(localStorage.getItem('contatos') || '[]');
Importante: Sempre use JSON.parse() para recuperar objetos e JSON.stringify() para salvar objetos.

4. Filtros e Busca com filter()

4.1 Conceito do filter()

O método filter() cria um novo array com elementos que passam em um teste. É perfeito para implementar busca e filtros.

4.2 Sintaxe Básica

// Sintaxe básica do filter()
const numeros = [1, 2, 3, 4, 5, 6];

const pares = numeros.filter(numero => numero % 2 === 0);
// Resultado: [2, 4, 6]

const maioresQue3 = numeros.filter(numero => numero > 3);
// Resultado: [4, 5, 6]

4.3 Implementando Busca no Projeto

5. Importância das Keys

5.1 Por que usar Keys?

As keys ajudam o React a identificar quais itens foram alterados, adicionados ou removidos. Sem keys, o React pode renderizar incorretamente ou perder o estado dos componentes.

5.2 Exemplo sem Key (❌ Problema)

// ❌ SEM key - pode causar problemas
{contacts.map((contact) => (
  <ContactItem contact={contact} />
))}

5.3 Exemplo com Key (✅ Correto)

// ✅ COM key - React pode otimizar
{contacts.map((contact) => (
  <ContactItem 
    key={contact.id} 
    contact={contact} 
  />
))}

5.4 Regras para Keys

  • Únicas: Cada key deve ser única entre irmãos
  • Estáveis: Não devem mudar entre re-renderizações
  • Previsíveis: Devem ser consistentes

5.5 Exemplos de Keys Boas e Ruins

// ✅ BOAS keys
key={contact.id}           // ID único do banco
key={`contact-${contact.id}`} // Prefixo + ID
key={contact.email}        // Email único

// ❌ RUINS keys
key={Math.random()}        // Muda a cada render
key={index}               // Pode mudar ordem
key={contact.name}        // Pode não ser único

6.4 Página de Detalhes Atualizada

Agora que temos persistência, vamos atualizar a página de detalhes para buscar os dados do contato pelo ID no localStorage:

1. Atualizar imports
// app/contact/[id]/page.js
"use client";
import { useState, useEffect } from 'react';
import { useParams, useRouter } from 'next/navigation';
2. Implementar busca no localStorage
const ContactDetailPage = () => {
  const params = useParams();
  const router = useRouter();
  const [contact, setContact] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    // Buscar contatos do localStorage
    const savedContacts = localStorage.getItem('contatos');
    if (savedContacts) {
      const contacts = JSON.parse(savedContacts);
      const foundContact = contacts.find(c => c.id === parseInt(params.id));
      
      if (foundContact) {
        setContact(foundContact);
      } else {
        // Contato não encontrado
        router.push('/');
      }
    } else {
      // Não há contatos salvos
      router.push('/');
    }
    setLoading(false);
  }, [params.id, router]);

  if (loading) {
    return (
      <div className="min-h-screen bg-gray-50 p-6">
        <div className="max-w-2xl mx-auto">
          <div className="bg-white shadow rounded-lg p-6">
            <p className="text-gray-600">Carregando...</p>
          </div>
        </div>
      </div>
    );
  }

  if (!contact) {
    return (
      <div className="min-h-screen bg-gray-50 p-6">
        <div className="max-w-2xl mx-auto">
          <div className="bg-white shadow rounded-lg p-6">
            <p className="text-gray-600">Contato não encontrado</p>
          </div>
        </div>
      </div>
    );
  }

  return (
    <div className="min-h-screen bg-gray-50 p-6">
      <div className="max-w-2xl mx-auto">
        <div className="bg-white shadow rounded-lg p-6">
          <div className="flex items-center justify-between mb-6">
            <h1 className="text-2xl font-bold text-gray-900">Detalhes do Contato</h1>
            <button
              onClick={() => router.back()}
              className="bg-gray-100 hover:bg-gray-200 text-gray-900 px-4 py-2 rounded"
            >
              ← Voltar
            </button>
          </div>

          <div className="space-y-4">
            <div>
              <label className="block text-sm font-medium text-gray-700 mb-1">Nome</label>
              <p className="text-gray-900">{contact.nome}</p>
            </div>
            
            <div>
              <label className="block text-sm font-medium text-gray-700 mb-1">Email</label>
              <p className="text-gray-900">{contact.email}</p>
            </div>
            
            <div>
              <label className="block text-sm font-medium text-gray-700 mb-1">Telefone</label>
              <p className="text-gray-900">{contact.telefone}</p>
            </div>
          </div>
        </div>
      </div>
    </div>
  );
};

export default ContactDetailPage;
Vantagens: Agora a página de detalhes busca os dados diretamente do localStorage, eliminando a necessidade de passar dados via URL e tornando a navegação mais limpa e eficiente!

Exercício 1: Lista de Tarefas

Objetivo: criar uma aplicação de lista de tarefas com persistência e filtros.

  1. Adicionar nova tarefa com título e prioridade
  2. Marcar tarefa como concluída
  3. Filtrar por prioridade (alta, média, baixa)
  4. Buscar tarefas por texto
  5. Salvar dados no localStorage
Mostrar solução 
// Estrutura de dados
const [tarefas, setTarefas] = useState([]);
const [filtro, setFiltro] = useState('');
const [filtroPrioridade, setFiltroPrioridade] = useState('todas');

// Carregar do localStorage
useEffect(() => {
  const saved = localStorage.getItem('tarefas');
  if (saved) setTarefas(JSON.parse(saved));
}, []);

// Salvar no localStorage
useEffect(() => {
  localStorage.setItem('tarefas', JSON.stringify(tarefas));
}, [tarefas]);

// Filtro por prioridade e texto
const tarefasFiltradas = tarefas.filter(tarefa => {
  const matchTexto = tarefa.texto.toLowerCase().includes(filtro.toLowerCase());
  const matchPrioridade = filtroPrioridade === 'todas' || tarefa.prioridade === filtroPrioridade;
  return matchTexto && matchPrioridade;
});
