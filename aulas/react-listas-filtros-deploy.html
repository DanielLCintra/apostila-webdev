2. Listas Dinâmicas com map()

2.1 Conceito do map()

O método map() é fundamental para renderizar listas em React. Ele transforma cada elemento de um array em um elemento JSX.

2.2 Sintaxe Básica

// Sintaxe básica do map()
const lista = [1, 2, 3, 4, 5];

const elementosJSX = lista.map((item, index) => {
  return <li key={index}>{item}</li>;
});

// Renderização
return <ul>{elementosJSX}</ul>;

2.3 Exemplo Prático - Lista de Contatos

// Componente ContactList
export default function ContactList({ contacts, onRemove }) {
  return (
    <div className="mt-6">
      <h3 className="text-lg font-semibold text-gray-900 mb-4">Lista de Contatos</h3>
      <ul className="space-y-2">
        {contacts.map((contact) => (
          <ContactItem 
            key={contact.id} 
            contact={contact} 
            onRemove={onRemove} 
          />
        ))}
      </ul>
    </div>
  );
}
Importante: Sempre use uma key única para cada elemento da lista. Isso ajuda o React a identificar e atualizar elementos eficientemente.

3. Persistência com localStorage

3.1 Conceito do localStorage

O localStorage é uma API do navegador que permite armazenar dados no navegador do usuário. Os dados persistem mesmo após fechar e reabrir o navegador.

3.2 Métodos Principais

// Salvar dados
localStorage.setItem('chave', 'valor');

// Recuperar dados
const valor = localStorage.getItem('chave');

// Remover dados
localStorage.removeItem('chave');

// Limpar tudo
localStorage.clear();

// Verificar se existe
if (localStorage.getItem('chave')) {
  // dados existem
}

3.3 Trabalhando com Objetos

Como o localStorage só armazena strings, precisamos converter objetos para JSON: 

// Salvar array de objetos
const contatos = [
  { id: 1, nome: 'João', email: 'joao@email.com' },
  { id: 2, nome: 'Maria', email: 'maria@email.com' }
];

localStorage.setItem('contatos', JSON.stringify(contatos));

// Recuperar array de objetos
const contatosSalvos = JSON.parse(localStorage.getItem('contatos') || '[]');
Importante: Sempre use JSON.parse() para recuperar objetos e JSON.stringify() para salvar objetos.

4. Filtros e Busca com filter()

4.1 Conceito do filter()

O método filter() cria um novo array com elementos que passam em um teste. É perfeito para implementar busca e filtros.

4.2 Sintaxe Básica

// Sintaxe básica do filter()
const numeros = [1, 2, 3, 4, 5, 6];

const pares = numeros.filter(numero => numero % 2 === 0);
// Resultado: [2, 4, 6]

const maioresQue3 = numeros.filter(numero => numero > 3);
// Resultado: [4, 5, 6]

4.3 Implementando Busca no Projeto

// Componente FilterInput

5. Importância das Keys

5.1 Por que usar Keys?

As keys ajudam o React a identificar quais itens foram alterados, adicionados ou removidos. Sem keys, o React pode renderizar incorretamente ou perder o estado dos componentes.

5.2 Exemplo sem Key (❌ Problema)

// ❌ SEM key - pode causar problemas
{contacts.map((contact) => (
  <ContactItem contact={contact} />
))}

5.3 Exemplo com Key (✅ Correto)

// ✅ COM key - React pode otimizar
{contacts.map((contact) => (
  <ContactItem 
    key={contact.id} 
    contact={contact} 
  />
))}

5.4 Regras para Keys

  • Únicas: Cada key deve ser única entre irmãos
  • Estáveis: Não devem mudar entre re-renderizações
  • Previsíveis: Devem ser consistentes

5.5 Exemplos de Keys Boas e Ruins

// ✅ BOAS keys
key={contact.id}           // ID único do banco
key={`contact-${contact.id}`} // Prefixo + ID
key={contact.email}        // Email único

// ❌ RUINS keys
key={Math.random()}        // Muda a cada render
key={index}               // Pode mudar ordem
key={contact.name}        // Pode não ser único

6.2 HomePage Atualizada com localStorage

// app/page.js - HomePage expandida (continuação da aula anterior)
"use client";
import { useState, useEffect } from 'react';
import ContactForm from '../components/ContactForm';
import ContactList from '../components/ContactList';
import FilterInput from '../components/FilterInput';

export default function HomePage() {
  const [contacts, setContacts] = useState([]);
  const [filter, setFilter] = useState('');

  // Carregar contatos do localStorage na inicialização
  useEffect(() => {
    const savedContacts = localStorage.getItem('contatos');
    if (savedContacts) {
      setContacts(JSON.parse(savedContacts));
    }
  }, []);

  // Salvar contatos no localStorage sempre que mudarem
  useEffect(() => {
    localStorage.setItem('contatos', JSON.stringify(contacts));
  }, [contacts]);

  const handleSubmit = (newContact) => {
    setContacts(prev => [...prev, { ...newContact, id: Date.now() }]);
  };

  const handleRemove = (id) => {
    setContacts(prev => prev.filter(c => c.id !== id));
  };

  // Filtro simples por nome ou email
  const filteredContacts = contacts.filter(contact =>
    contact.nome.toLowerCase().includes(filter.toLowerCase()) ||
    contact.email.toLowerCase().includes(filter.toLowerCase())
  );

  return (
    <div className="min-h-screen bg-gray-200 p-6">
      <div className="max-w-3xl mx-auto space-y-6">
        <header className="flex items-center justify-between">
          <h1 className="text-2xl font-bold text-gray-900">Cadastro de Contatos</h1>
          <FilterInput value={filter} onChange={setFilter} />
        </header>

        <ContactForm onAdd={handleSubmit} />

        <ContactList items={filteredContacts} onRemove={handleRemove} />
      </div>
    </div>
  );
}

      </button>
