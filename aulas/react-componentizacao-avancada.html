React – Componentização Avançada e Composição

Neste encontro vamos aprofundar as técnicas de componentização, explorando diferentes formas de compor interfaces e evitar problemas comuns como o props drilling.

  • ✅ Técnicas de composição e reutilização
  • ✅ Uso avançado do children
  • ✅ Introdução ao problema do props drilling

1. Técnicas de Composição

Componentes menores podem ser combinados para formar estruturas mais complexas. Essa abordagem facilita o reuso e a leitura do código. 

function Toolbar() {
  return (
    <div className="toolbar">
      <Botao texto="Salvar" />
      <Botao texto="Cancelar" />
    </div>
  );
}

2. Uso do children

A prop especial children permite que um componente envolva elementos arbitrários, tornando-o altamente reutilizável. 

function Painel({ children }) {
  return <section className="painel">{children}</section>;
}

export default function App() {
  return (
    <Painel>
      <h2>Título</h2>
      <p>Conteúdo do painel</p>
    </Painel>
  );
}

3. Props Drilling

O props drilling ocorre quando uma prop precisa ser passada por muitos níveis de componentes apenas para alcançar um componente mais interno. Isso torna o código difícil de manter.

Uma alternativa é utilizar o Context API ou bibliotecas de gerenciamento de estado, que veremos em aulas futuras.

4. Boas Práticas

  • Mantenha os componentes pequenos e com uma responsabilidade clara.
  • Organize os arquivos em pastas que reflitam a estrutura da aplicação.
  • Prefira composição em vez de herança.

5. Resumo

  • Componentes podem ser combinados para formar interfaces complexas.
  • children torna componentes mais flexíveis.
  • Evite o props drilling para manter o código limpo.

  1. Crie três componentes aninhados.
