Aula 1: Introdução ao React – Setup e Primeiros Componentes

React é uma biblioteca JavaScript para construir interfaces de usuário. Nesta aula iniciaremos um projeto com Next.js e criaremos nossos primeiros componentes.

  • Configurar o ambiente com npx create-next-app@latest
  • Entender a estrutura de pastas padrão de um projeto React
  • Criar os primeiros componentes funcionais utilizando JSX
  • Explorar os arquivos gerados pelo setup
  • Criação de um componente “Hello World” e visualização no navegador

1. Configuração do Projeto

Certifique-se de ter o Node.js instalado. Crie o projeto usando Next.js:

Terminal
npx create-next-app@latest meu-app

Depois entre na pasta e inicie o servidor de desenvolvimento:

cd meu-app
npm run dev

Principais pastas criadas:

  • pages/ ou app/ – páginas da aplicação
  • public/ – arquivos estáticos
  • styles/ – arquivos de estilo

2. Criando um Componente

No React, componentes são funções que retornam elementos JSX.

function Hello() {
  return <h1>Hello World</h1>;
}

Salve este componente em pages/index.js (ou app/page.js) para visualizá-lo no navegador.

Um componente pode receber props para tornar sua renderização dinâmica:

function Hello(props) {
  return <h1>Olá, {props.nome}!</h1>;
}

3. Resumo

  • Criamos um projeto com create-next-app@latest.
  • Exploramos a estrutura de pastas e iniciamos o servidor com npm run dev.
  • Construímos componentes funcionais simples e utilizamos props.

4. Exercícios

Exercício 1: Criar o Projeto

Objetivo: Instalar o projeto e rodar o servidor.

  1. Execute npx create-next-app@latest meu-app.
  2. Entre na pasta e rode npm run dev.
  3. Acesse http://localhost:3000 no navegador.

Exercício 2: Componente Hello World

Objetivo: Criar um componente que mostre "Hello World".

  1. Abra pages/index.js (ou app/page.js).
  2. Substitua o conteúdo pelo componente Hello mostrado acima.
  3. Salve e observe a mensagem no navegador.
