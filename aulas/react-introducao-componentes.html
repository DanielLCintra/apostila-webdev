Aula 1: Introdução ao React – Setup e Primeiros Componentes

React é uma biblioteca JavaScript para construir interfaces de usuário. Nesta aula iniciaremos um projeto com Next.js e criaremos nossos primeiros componentes.

  • Configurar o ambiente com npx create-next-app@latest
  • Entender a estrutura de pastas padrão de um projeto React
  • Criar os primeiros componentes funcionais utilizando JSX
  • Explorar os arquivos gerados pelo setup
  • Criação de um componente “Hello World” e visualização no navegador

1. O que é o React?

React é uma biblioteca JavaScript criada por Jordan Walke no Facebook em 2011 para facilitar a construção de interfaces dinâmicas. O projeto foi aberto ao público em 2013 e rapidamente ganhou popularidade por sua abordagem baseada em componentes e atualização eficiente da tela.

2. O que é JSX?

JSX é uma extensão de sintaxe do JavaScript criada pela equipe do React para descrever a interface de maneira declarativa. Ele permite escrever marcação semelhante a HTML dentro do código JavaScript e é transformado para chamadas React.createElement por ferramentas como o Babel.

Utilizar JSX torna o código mais legível e expressivo, pois a lógica e a estrutura visual ficam no mesmo arquivo. Embora não seja obrigatório, é largamente adotado na comunidade React pela praticidade.

3. Node.js e npm

Node.js é um ambiente de execução JavaScript fora do navegador.

Já o npm é o gerenciador de pacotes que acompanha o Node e facilita a instalação de bibliotecas.

Com eles configuramos projetos rapidamente e organizamos dependências como o React.

Também é possível utilizar React sem essas ferramentas, incluindo os scripts via CDN em uma página HTML:

Exemplo com import

index.html

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8" />
  <title>React sem Build</title>
</head>
<body>
  <div id="root"></div>
  <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
<script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
  <script src="script.js"></script>
</body>
</html>

script.js

function App() {
  return React.createElement('h1', null, 'Hello World sem build');
}

ReactDOM.render(
  React.createElement(App),
  document.getElementById('root')
);

Nesse cenário continuamos sem JSX, por isso utilizamos React.createElement.

5. Configuração do Projeto

Para iniciar um projeto com React usando Next.js, siga os passos abaixo. Certifique-se de ter o Node.js e o npm instalados em sua máquina.

Terminal
npx create-next-app@latest meu-app

Ao rodar esse comando, algumas perguntas serão feitas. Responda da seguinte forma:

Perguntas e Respostas
  • Would you like to use TypeScript? → No
  • Would you like to use ESLint? → Yes
  • Would you like to use Tailwind CSS? → No
  • Would you like to use `src/` directory? → No
  • Would you like to use App Router? (recommended) → Yes
  • Would you like to customize the default import alias? → No

Depois da instalação, entre na pasta do projeto e inicie o servidor de desenvolvimento:

cd meu-app
npm run dev

Acesse o navegador no endereço http://localhost:3000 para ver a aplicação funcionando.

6. Criando um Componente

Um componente é um bloco de código que representa uma parte da interface (UI) da sua aplicação. Em React, componentes são geralmente funções que retornam um conteúdo JSX (semelhante ao HTML), e podem ser reutilizados em várias partes do seu projeto.

Por exemplo, podemos criar um componente chamado Hello que exibe uma mensagem na tela.

Passo 1: Criando o projeto React

Se ainda não criou um projeto React, abra o terminal e execute o seguinte comando: 

npx create-next-app@latest meu-app

Depois, entre na pasta do projeto:

cd meu-app

Passo 2: Criando um componente simples com valor fixo

Agora, dentro da pasta src, crie um novo arquivo chamado Hello.js. Esse será o nosso primeiro componente.

Adicione o seguinte código no arquivo Hello.js:

function Hello() {
  return <h1>Hello World</h1>;
}

export default Hello;

Esse código cria uma função chamada Hello que retorna um título com o texto "Hello World". A linha export default serve para permitir que esse componente seja usado em outros arquivos.

Passo 3: Usando o componente na aplicação

No arquivo src/App.js, substitua o conteúdo da função App para usar o componente Hello: 

import Hello from './Hello';

function App() {
  return (
    <div>
      <Hello />
    </div>
  );
}

export default App;

Com isso, ao iniciar sua aplicação, o conteúdo do componente Hello será exibido na tela.

Passo 4: Tornando o componente dinâmico com props

Podemos melhorar nosso componente para que ele receba um nome como parâmetro. Esses parâmetros são chamados de props em React.

Atualize o conteúdo do Hello.js para:

function Hello(props) {
  return <h1>Olá, {props.nome}!</h1>;
}

export default Hello;

Agora o componente espera receber uma prop chamada nome, e a utiliza para montar a frase.

Passo 5: Enviando props para o componente

No arquivo App.js, envie um nome para o componente usando o seguinte código: 

import Hello from './Hello';

function App() {
  return (
    <div>
      <Hello nome="Maria" />
      <Hello nome="João" />
    </div>
  );
}

export default App;

Agora você verá na tela:

  • Olá, Maria!
  • Olá, João!

Cada vez que você usar o componente <Hello /> com um nome diferente, ele exibirá uma mensagem personalizada.

Resumo

  • Componentes são funções que retornam JSX.
  • Você pode reutilizar um componente em várias partes da aplicação.
  • As props permitem deixar os componentes dinâmicos.
  • Você pode criar componentes em arquivos separados e importá-los onde quiser.

Essa estrutura modular é uma das principais forças do React!

8. Exercícios

Exercício 1: Criar o Projeto

Objetivo: Instalar o projeto Next.js e rodar o servidor de desenvolvimento.

  1. No terminal, execute npx create-next-app@latest meu-app.
  2. Responda às perguntas conforme indicado na seção anterior.
  3. Entre na pasta criada com cd meu-app.
  4. Inicie o servidor com npm run dev.
  5. Acesse http://localhost:3000 no navegador e confirme que a aplicação está rodando.

Exercício 3: Componente com props

Objetivo: Criar um componente que exiba uma saudação personalizada com base em uma prop.

  1. Crie um novo arquivo chamado Hello.js.
  2. Crie um componente que receba uma prop chamada nome e mostre: Olá, {nome}!.
  3. No arquivo page.js, importe e utilize o componente passando diferentes nomes.
  4. Visualize no navegador a saudação personalizada para cada nome.
Ver solução

Arquivo Hello.js:

function Hello({ nome }) {
  return <h1>Olá, {nome}!</h1>;
}
export default Hello;

Arquivo page.js:

import Hello from './Hello';

export default function Home() {
  return (
    <div>
      <Hello nome="Maria" />
      <Hello nome="João" />
    </div>
  );
}
