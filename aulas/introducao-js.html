1. História e Evolução do JavaScript

1995

Criado por Brendan Eich na Netscape em apenas 10 dias.

Inicialmente chamado de Mocha, depois LiveScript, e finalmente JavaScript.

1997

JavaScript é padronizado pela ECMA International como ECMAScript.

2009

ES5 introduz importantes adições como JSON.parse e strict mode.

2015

ES6/ES2015 traz let/const, arrow functions, classes, promises e muito mais.

2016-2024

Lançamentos anuais com novas funcionalidades.

JavaScript se expande para servidores (Node.js), aplicações móveis e desktop.

JavaScript evoluiu de uma linguagem simples para uma plataforma completa de desenvolvimento:

  • Navegadores: De pequenos scripts a aplicações web completas (SPAs)
  • Servidores: Node.js permitiu JavaScript no backend
  • Mobile: React Native, Ionic
  • Desktop: Electron
  • IoT e dispositivos: JavaScript em diversos dispositivos

2. JavaScript no Navegador

Os navegadores modernos executam JavaScript através de motores altamente otimizados:

  • V8: Desenvolvido pelo Google, usado no Chrome, Edge e Node.js
  • SpiderMonkey: Usado no Firefox
  • JavaScriptCore: Usado no Safari

Características importantes incluem:

  • APIs do navegador: DOM, Window, Fetch, localStorage
  • Runtime único (single-threaded) com event loop
  • Interação com HTML e CSS para manipulação da página

3. Inclusão de JavaScript em HTML

Existem diferentes formas de incluir JavaScript em uma página:

Script interno no HEAD ou BODY
<script>
  console.log("Script interno");
</script>
Script externo
<script src="script.js"></script>
Com atributo defer (carrega em paralelo, executa após o parsing do HTML) 
<script src="script.js" defer></script>
Com atributo async (carrega e executa de forma assíncrona) 
<script src="script.js" async></script>

4. Primeiros Passos com JavaScript

HTML, CSS e JavaScript: A Analogia do Corpo Humano

Pense nas tecnologias web como partes de um corpo humano:

  • HTML (Esqueleto): Assim como o esqueleto define a estrutura básica do corpo humano, o HTML define a estrutura fundamental de uma página web. Ele determina os elementos essenciais, como títulos, parágrafos, imagens e links - os "ossos" do seu website.
  • CSS (Pele e Aparência): Tal como a pele dá cor, textura e forma ao corpo, o CSS é responsável pela aparência visual do site. Ele controla cores, fontes, espaçamentos, layout e design - "vestindo" o esqueleto HTML com estilo e elegância.
  • JavaScript (Músculos e Movimento): Os músculos permitem movimento e interação no corpo humano, assim como o JavaScript adiciona interatividade e dinamismo a uma página web. Ele permite que os elementos respondam a eventos, se movam, mudem e interajam com o usuário.

5. Console para Debugging

O console é uma ferramenta essencial para desenvolvimento:

Métodos do Console
// Métodos básicos
console.log("Mensagem normal");
console.info("Informação");
console.warn("Aviso");
console.error("Erro");

// Tabelas
console.table([
  {id: 1, tarefa: "Estudar JS"},
  {id: 2, tarefa: "Praticar código"}
]);

// Agrupamento
console.group("Grupo de logs");
console.log("Log 1");
console.log("Log 2");
console.groupEnd();

// Medição de tempo
console.time("Timer");
// código a ser medido
console.timeEnd("Timer");

7. Case Sensitivity

JavaScript é case-sensitive, o que significa que diferencia letras maiúsculas e minúsculas:

Exemplo de Case Sensitivity
// JavaScript é case-sensitive
let nome = "TaskMaster";
let Nome = "Outro nome";
console.log(nome); // TaskMaster
console.log(Nome); // Outro nome

8. Variáveis em JavaScript

Variáveis são utilizadas para armazenar dados que podem ser usados em seu código:

Declaração de Variáveis
// Exemplos básicos de variáveis
let appVersion = 0.1;
let isActive = true;

// Mostrando valores no console
console.log("Versão do aplicativo:", appVersion);
console.log("Aplicativo ativo:", isActive);

9. Trabalhando com Strings

Existem diferentes formas de manipular strings em JavaScript:

Concatenação Tradicional
// Exemplo simples de concatenação de strings
let mensagem = "Olá! Bem-vindo ao " + nome;
console.log(mensagem);
Template Strings (ES6+)
// Exemplo de template string (forma moderna de concatenação)
let mensagemTemplate = `Aplicativo: ${nome} - Versão: ${appVersion}`;
console.log(mensagemTemplate);

10. Alertas no Navegador

Alertas são caixas de diálogo que podem ser usadas para interagir com o usuário:

Exemplos de Alert
// Exemplo de alert simples
// alert("Bem-vindo ao TaskMaster!");

// alert(mensagemAlert);

Exercício 5: Debugging Básico

Copie e cole o seguinte código e observe o resultado no console:

let numero1 = 10;
let numero2 = "5";
console.log(numero1 + numero2);

O resultado é 105, pois o JavaScript está concatenando ao invés de somar.

Ver solução
Solução
let numero1 = 10;
let numero2 = "5";
console.log(numero1 + Number(numero2)); // ou parseInt(numero2)

// Outra solução:
let numero1 = 10;
let numero2 = 5; // Modificar diretamente para um número
console.log(numero1 + numero2);
